## Git command 정리
  
### 브런치명 변경
```bash
git branch -m [target_branch] [branch new name]
```

### 브런치 생성 & 변경
```bash
git checkout -b [branch name]
```

### 브런치 생성만
```bash
git branch [branch name]
```

### git config (local)
```bash
git config --local user.email "example@examples.com"
git config --local user.name "example"
```
